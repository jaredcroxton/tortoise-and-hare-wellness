# Tortoise and Hare Wellness

Landing page for the AI Wellness Coach — a psychology-grounded AI companion that hears what your team isn't saying, flagging psychological safety and burnout before they break.

Grounded in psychology. Designed for real results.

## Stack

Static single-page site. No build step.

## Run locally

```bash
python3 -m http.server 4173
# then open http://localhost:4173
```

## Deploy

Hosted on Vercel. Any push to `main` deploys automatically.

Learn more: [tortoiseandharewellness.com](https://tortoiseandharewellness.com)
